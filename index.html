Boids

Boids is an artificial life program that simulates the behaviour of birds, fish, and swarms. The boids follow a set of rules that demonstrate complex emergence behaviour when interacting with other boids within their field of view. This project was created entirely from scratch, meaning that we created our own Vector and Utility libraries in order to perform all the physics, vector operations, and trigonometry needed to simulate the boid movement.

Yelp Data Search Appication

The Yelp Data Search Application allows users to query a large dataset and extract information about local businesses and individual users. The dataset used in this project contains 20,000 businesses, 190,000 users, 290,000 tips, and 3.8 million check-ins to those businesses. I parsed the dataset (JSON) using Python, designed an Entity-Relationship model and translated it into a Relational schema. I wrote Python scripts to populate our database and wrote complex queries for our standalone application's search functions using SQL and C#.
